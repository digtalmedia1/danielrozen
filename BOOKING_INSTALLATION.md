# מדריך התקנת מערכת הזמנות V2 - דניאל רוזן

מערכת זו נבנתה עבור Hostinger עם PHP 8+ ו-MySQL.

## שלבי התקנה

1. **העלאת קבצים:**
   העלה את כל תוכן התיקייה לשרת ה-FTP שלך.

2. **הגדרת מסד נתונים:**
   - צור מסד נתונים MySQL חדש ב-Hostinger.
   - ייבא את הקובץ `install.sql` דרך phpMyAdmin.

3. **הגדרת סביבה:**
   - צור קובץ `.env` בתיקייה הראשית (מבוסס על `.env.example`).
   - הזן את פרטי מסד הנתונים.
   - קבע `INSTALL_TOKEN` חזק וסודי.

4. **יצירת מנהל ראשון:**
   - גלוש לכתובת: `https://yourdomain.com/install/create-admin.php?token=YOUR_TOKEN`
   - מלא את הטופס ליצירת המשתמש.
   - **חשוב:** לאחר הסיום, מחק את תיקיית `/install/` מהשרת.

5. **הגדרת Cron Job (אופציונלי אך מומלץ):**
   בפאנל Hostinger, הגדר Cron שיריץ את הפקודה הבאה כל 15 דקות לניקוי הזמנות שפגו:
   `php /home/uXXXXX/public_html/includes/cron_expire.php`

## אבטחה
- המערכת משתמשת ב-CSRF Protection.
- סיסמאות נשמרות בצורה מוצפנת (Bcrypt).
- קיימת הגנה מפני Brute Force.
- כל הקבצים המנהליים דורשים התחברות.
